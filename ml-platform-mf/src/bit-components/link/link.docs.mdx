import {Link} from './link'
import {style} from './utils'

The Link component allows you to easily customize anchor elements.

### Component usage

```js
import {Link} from '@sporta-technology/d11-web-components.link'

<Link href='#' underline='none'>
  Click me!
</Link>
```

### Render Link!

```js live
<div style={style}>
  <Link href='#' underline='none'> Click me!</Link>
</div>
```
