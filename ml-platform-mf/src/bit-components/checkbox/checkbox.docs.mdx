import {Checkbox} from './checkbox'

Checkbox allow users to select multiple items from a list of individual items.

### Component usage

```js
import {Checkbox,CheckboxProps} from '@sporta-technology/d11-web-components.checkbox'

<Checkbox checked={checked} name='test' onChange={handleChange} />
```

### Render Checkbox !

```js live
<div style={style}>
  <Checkbox name='test' text='Checkbox' />
</div>
```
