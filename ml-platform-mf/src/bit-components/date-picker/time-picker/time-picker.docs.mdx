import {TimePicker} from './time-picker'

Time pickers picker allow selecting a single value from a pre-determined set and you can also pass a time.

### Component usage

```js
import {DatePicker} from '@sporta-technology/d11-web-components.date-picker.time-picker'
;<TimePicker label='Label' value={value} onChange={handleChange} />
```
