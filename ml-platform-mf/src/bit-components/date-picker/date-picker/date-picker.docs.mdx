import {DatePicker} from './date-picker'

Date pickers picker allow selecting a single value from a pre-determined set and you can also pass a valid date.

### Component usage

```js
import {DatePicker} from '@sporta-technology/d11-web-components.date-picker.date-picker'
;<DatePicker label='Label' value={value} onChange={handleChange} />
```

```js live
<DatePicker label='Date' onChange={() => {}} />
```
