import {DateTimePicker} from './date-time-picker'

Date-Time picker allow selecting a single value from a pre-determined set and you can also pass a valid date-time.

### Component usage

```js
import {DateTimePicker} from '@sporta-technology/d11-web-components.date-picker.date-time-picker'
;<DateTimePicker label='Label' value={value} onChange={handleChange} />
```

### Render Date Time Picker!

```js live
<DateTimePicker label='Date time picker' onChange={() => {}} />
```
