import {Typography} from './typography'

A component is used to display the text and strings.
To use Roboto font family , you have to install roboto explicity in your application

### Component usage

```js
<Typography>Hello world!</Typography>
```

### Render Typography component!

```js live
<Typography sx={{fontSize: '14px'}} variant='body1'>
  Hello world!
</Typography>
```
