import {Tooltip} from './tooltip'
import {ToolTipPlacement} from './constants'
import Button from '@mui/material/Button'

Tooltips show contextual help or information about specific components when a user hovers

### Component usage

```js
 <Tooltip title='Text' placement={ToolTipPlacement.BottomStart}>
        <Button>Tool Tip Text</Button>
      </Tooltip>
```

### Render Tooltip component!

```js live
 <Tooltip title='Text' placement={ToolTipPlacement.BottomStart}>
        <Button>Tool Tip Text</Button>
      </Tooltip>
```
