import {BitThemeWrapper} from './bit-theme-wrapper'

A component that does something special and renders text in a div.

### Component usage

```js
<BitThemeWrapper theme="light"> <App> </BitThemeWrapper>
<BitThemeWrapper theme="dark"> <App> </BitThemeWrapper>
```

### Render hello world!

```js live
<BitThemeWrapper theme='dark'>Hello world!</BitThemeWrapper>
```
