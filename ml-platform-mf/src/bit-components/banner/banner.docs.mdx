A component is used to alert and messages in a form of banner inside a modal.

### Component usage

```js
import {
  Banner,
  BannerProps,
  Severity
} from '@sporta-technology/d11-web-components.banner'
import {Icons} from '@sporta-technology/d11-web-components.icon'
;<Banner messages={'Dummy Text for Banner Component'} />
```

```js live
<div
  style={{

  }}
>
  <Banner
    message={'Dummy Text for Banner Component'}
```
